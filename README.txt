git 사용법

1. github에 repository를 생성한다.
2. 생성한 repository를 clone한다 (git clone https://github.com/id/어쩌고.git)
3. 폴더에서 작업을 한다.
4. git status / git diff 로 변경 내역을 본다 
5. git add --all 로 변경 내역을 추가한다.
6. git commit 으로 변경 내역을 고정시킨다 (git commit -m "변경 내역")
7. git push 를한다.
8. 변경된 점이 있는지 git pull 명령어를 친다.
